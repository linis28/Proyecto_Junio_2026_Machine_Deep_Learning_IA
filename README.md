# Machine Learning - Apuntes y Prácticas

##  Descripción

Este repositorio contiene apuntes, resúmenes y ejercicios prácticos basados en un libro de Machine Learning.  
El objetivo es comprender los fundamentos teóricos y su implementación práctica en Python.

---

##  Objetivos del proyecto

- Comprender los fundamentos del Machine Learning
- Estudiar algoritmos supervisados y no supervisados
- Implementar modelos básicos en Python
- Practicar con datasets reales

---

##  Contenido

- Regresión lineal y logística  
- Árboles de decisión  
- K-Nearest Neighbors (KNN)  
- Clustering (K-Means)  
- Evaluación de modelos  
- Overfitting y underfitting  

---

## Tecnologías usadas

- Python 3.x  
- NumPy  
- Pandas  
- Scikit-learn  
- Matplotlib / Seaborn
